# dealwithit

To display, run:
```
vm.displayAt = {top: 30, left: 40};
```

To hide, click the shades

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
